git is Used for version control

git init - initialize repository
git status - check if something has been added, untracked, staged, unstaged
git add - stage a change
git commit -m  '[descriptive message]' - save staged changes

git log - shows all commits

git branch - shows all the branches 
git branch -b [branch name]- adds a branch to the current commit
git checkout [branch name] - goes to that branch
git checkout [id] - go to a specific commit

git remode add [name repository] [place url of external repository here]
git fetch [name of repository] - grabs all branches in that repository
git push [name of repository] [branch name currently on] - sends your current branch to the repository specified
git clone [place url of repository]
